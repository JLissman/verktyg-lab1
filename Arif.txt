* Vilka filer bör finnas i .gitignore? 

1. Log files
2. Files with API key/secrets, credentials, or sensitive information
3. Useless system files like .DS_Stroe on macOS

* Vad är en applikationsserver och varför behövs den?

An application server is a mixed framework of software tha allows both the creation of web applications and a server environment to run them.
An applications server is designed to install, operate and host applications and accociated services for end uses.
